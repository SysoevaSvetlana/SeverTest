# Система приемки поставок

## Описание кейса

В системе имеется **3 поставщика**, каждый из которых может поставлять:
- 2 вида груш
- 2 вида яблок

Поставщики заранее задают **цены на виды продукции** на определённый период поставок.

Приложение реализует:
- приёмку поставок от поставщиков
- формирование отчётов по поставкам за выбранный период


---

## Задачи

### 1. Приёмка поставок
- Поставка создаётся от конкретного поставщика
- В одной поставке может быть **несколько видов продукции**
- Для каждой позиции указывается вес
- Стоимость рассчитывается автоматически на основе цены, действующей на дату поставки

### 2. Отчётность
- Формирование отчёта за выбранный период
- Отображение:
  - поставщика
  - вида продукции
  - общего веса
  - общей стоимости

---


## Авторизация и роли

Реализована **JWT-аутентификация** с ролями пользователей:

- `SUPPLIER` — поставщик
- `BUYER` — покупатель
  

Доступ к API ограничен ролями на уровне Spring Security.
